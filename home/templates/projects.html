Solar System Simulator

The Solar System Simulator is a coded implementation of Newtons gravity formula except the gravitational constant as 1 for ease of use. I built this project in order to demonstrate my software development and GUIX interface development skills (with PyQt5).

Pathfinding Visualizer

The project I'm most proud of, my Pathfinding Visualizer uses the pygame standard library (for animation building) and the Tkinter module (for interface development). It allows you to generate a grid with your custom dimensions, and populate it with walls. You set a start and end point and select which of the Pathfinding algorithms you'd like to use: Djkistras' (basically BFS) or A* (hueristics approach). The algorithm will than find the shortest path from the start node to the end, and you'll be able to visualize it as it works as well.

Discord Bot

A Discord Bot which I made while working free-lance for a tournament hosting organization. This project, unlike the others, does not have a download button, but instead contains a link to a server in which this Bot exists. Type !help for help!

Cipher Solver

I built the cipher solver as a kind of pet project because of my fascination with codes and ciphers, however I decided to add it to my resume because I believe it demonstrates problem solving and data manipulation skills as well. The solver allows you to choose one of five different excryption methods and than encrypt or decrypt text based on that method using a secret key that you provide.
